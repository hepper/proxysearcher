Product description

Burd's Proxy Searcher enables you to find public proxy servers that you can use for anonymous web surfing. The program uses a customizable search engine query that retrieves a list of proxy servers and then tests the results for connection speed and availability.
Advanced users can tweak the query settings to their preference and also add custom searches for specific proxy types, but if you just want to find free HTTP proxies, there is no need to tinker with the settings.
Proxy Searcher displays a list of proxies that it found, along with a color coded status indicator, country and response times. If you are using supported browsers, you can apply the proxy settings with the click of a button, other browsers need to be configured manually.

Requirements

  • Windows operation system (tested on Windows 7 32 bit).
  • RAM 1GB.
  • Internet channel at least 1Mb/s.
